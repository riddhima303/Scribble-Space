# Scribble-Space 

Scribble-Space is a playful pixel-art drawing application built with React. Users can draw freely on a graph-paper canvas, choose different pen colors, erase sketches, and interact with a cute cat companion.

## Features

* 🎨 Freehand drawing on a canvas
* 🩹 Eraser tool
* 🐱 Cat popup on startup
* 🐾 Random cat placement on the screen
* 🗑️ Clear canvas
* ✨ Pixel-art inspired interface

## Tech Stack

* React
* JavaScript
* HTML5 Canvas
* CSS

## Installation

git clone https://github.com/riddhima303/Scribble-Space.git
cd Scribble-Space
npm install
npm run dev

## How to Use

1. Open the application.
2. Click **OK** to receive your cat.
3. Select a color from the palette.
4. Draw anywhere on the canvas.
5. Use the eraser to remove drawings.
6. Use the clear button to reset the canvas.
